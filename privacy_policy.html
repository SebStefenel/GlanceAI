GlanceAI Privacy Policy

Last updated: July 2026

GlanceAI is a Chrome extension that generates AI summaries of Google Search results. This policy explains what data is involved when you use it.

Data We Do Not Collect

GlanceAI does not collect, store, or transmit any data to its developer. There are no analytics, no tracking, no user accounts, and no servers operated by GlanceAI.

Your API Key

GlanceAI requires a personal API key from Groq. This key is stored locally on your device using Chrome's built-in chrome.storage.sync, which may sync across your Chrome profile. Your API key is never sent anywhere other than directly to Groq's API servers to authenticate requests you initiate.

Your Preferences

Your chosen summary format (e.g. bullet points, pros & cons) is stored locally using chrome.storage.sync. This is a user preference only and is never transmitted to any server.

Article Content

When you activate GlanceAI and hover over a search result link, the extension fetches that article and sends a portion of its text (up to approximately 2,750 characters) along with your search query to the Groq API. This is done solely to generate the summary you requested. GlanceAI does not store, log, or retain any of this content beyond the current browser session.

Generated summaries are cached in memory for the duration of your browser session so that hovering the same link twice does not send a duplicate request. This cache exists only in RAM and is cleared when the browser or tab is closed.

Third-Party Service: Groq

AI summaries are generated by Groq's API. Text sent to Groq is subject to Groq's Privacy Policy. You are responsible for reviewing Groq's terms before use.

Permissions

GlanceAI requests access to all URLs (<all_urls>) solely to fetch article content from whichever website a search result links to. Google Search results can point to any domain on the internet, so it is not possible to enumerate specific sites in advance. No data is accessed beyond what is needed to generate summaries for links you explicitly hover over while the extension is active.

Contact

If you have questions about this policy, please open an issue on the extension's GitHub repository.
